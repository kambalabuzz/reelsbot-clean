# Background Music

Place your royalty-free background music files here.

## Supported Formats
- `.mp3`
- `.wav`
- `.m4a`

## Recommended Sources for Royalty-Free Music
1. **YouTube Audio Library** - https://studio.youtube.com/channel/UC.../music
2. **Pixabay Music** - https://pixabay.com/music/
3. **Incompetech** - https://incompetech.com/music/royalty-free/
4. **Bensound** - https://www.bensound.com/
5. **Free Music Archive** - https://freemusicarchive.org/

## Usage
The video assembly will randomly select one music track from this directory and loop it as background music at 10% volume.

If no music files are present, videos will be generated without background music (voice only).
